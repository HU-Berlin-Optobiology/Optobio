What it does: Extracts given pages from provided PDF and saves them in another PDF in given path under given name
Very low key User-Interface; Needs Python3 installed on the machine to run the .py-file.
What to do:
1st row: Enter path to the PDF-file to extract pages from; Button can be used to navigate to said PDF.
2nd row: Enter path + filename for to be generated PDF-file; Button can be used to navigate to the path and to enter name of the file.
3rd row: Enter pagenumber(s) seperated with "," that you want have extracted. Click "Extract" button to extract entered pagenumber(s).
Once the entered numbers dissappear, the PDF-file should be generated.
